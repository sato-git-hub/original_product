# 要件定義
## 一言サービスコンセプト（サービスのキャッチコピーを一言で）
イラストレーター向けマッチングアプリ
## 誰のどんな課題を解決するのか？
自分の作品がどのくらい閲覧されているか確認することができる
クリエーター側は自身の作品と似たクリエーターを見つけることができる
閲覧ユーザーは自分の好みと似たユーザーとマッチングすることができる
## なぜそれを解決したいのか？
自身の経験から、自分と似た感性を持った人とコミュニケーションをとってみたいと思ったため
## どうやって解決するのか？
レコメンド機能の実装
- 作品ごとにLikeしたユーザーが表示される
- ユーザーごとにLikeした作品一覧が表示される
