# 要件定義
## 一言サービスコンセプト（サービスのキャッチコピーを一言で）
自分好みのイラストに出会えるマッチングアプリ
## 誰のどんな課題を解決するのか？
自分の感性に合ったイラストと出会うことができるため、以下のような使い方ができる
- クリエーター側は自身の作品と似たイラストレーターを見つけることができる
- 閲覧ユーザーは自分の好みと似たイラストとの出会い、またそのイラストレータを好む他ユーザと出会うことができる
## なぜそれを解決したいのか？
私自身が、自分と似た感性を持った人とコミュニケーションをとってみたいという思いがあるため
## どうやって解決するのか？
レコメンド機能の実装
- 作品ごとにLikeしたユーザーが表示される
- ユーザーごとにLikeした作品一覧が表示される
