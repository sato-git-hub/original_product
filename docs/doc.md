# オリジナルプロダクトのテーマ

## 「解決したいニーズ」 
自分好みの作品に出会う
## 「対象顧客」 
イラストを見ることを趣味としている人　イラストレーター　イラスト作品の購入を考えている人
## 「プロダクト名」 
ReCo
## 「プロダクトのカテゴリー」 
SNS投稿サイト
## 「顧客が得られる利益や体験」 
- 様々なイラストレーターの作品を見ることができる
- ユーザーのいいね履歴をレコメンドの学習データとして与えることでユーザーの好みにあった作品に出会いやすくなる
- スライド形式に作品が表示されることで、直感的な操作性、ストレスの少なく利用することができる
- イラストレーター側のメリットとして、自身の作品を好んでいる人が他にどんな作品を好んでいるか知ることができる
- 閲覧ユーザー側のメリットとして、自分の好みと合った他ユーザーと出会うことができる
## 「他の主要な競合製品」 「競合との主要な違い」 
- Instagram イラスト作品を投稿することに特化したSNSではない
- Pinterest 投稿の大部分は、作品のクリエーター自身が投稿したものではなく引用されたもの
- Pixiv ユーザーが閲覧する作品を選択する必要がある
