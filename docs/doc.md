# オリジナルプロダクトのテーマ

## 「解決したい課題/潜在的なニーズ」 
クリエーターの作品が多くの人の目に触れやすいような仕組みを作りたい
## 「対象顧客」 
イラストレーター
## 「プロダクト名」 
LET REC
## 「プロダクトのカテゴリー」 
SNS投稿サイト
## 「顧客が得られる利益や体験」 
- 様々なイラストレーターの作品を見ることができる
- スライド形式で作品が表示されるため、直感的でストレスの少ない操作で作品を閲覧することができる
- レコメンド機能により、表示される作品は閲覧しているユーザーごとの嗜好にあったものを自動で選択するようになっており、閲覧するユーザーにとって、選択するというストレスがない
- このアプリを閲覧した他ユーザから人気のある作品を確認することができる
## 「他の主要な競合製品」 「競合との主要な違い」 
- Instagram クリエーターの作品を投稿することに特化したSNSではない
- Pinterest 作品のクリエーター自身が投稿する形ではない
- Pixiv ユーザーが閲覧する作品を選択する必要がある
