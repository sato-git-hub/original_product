# オリジナルプロダクトのテーマ

## 「解決したいニーズ」 
スキルはあるのに知名度の少ないイラストレーターを支援できる活動の場をつくる
## 「対象顧客」 
* イラストの完成度や技術力は高いが、イラストレーター独自のコンテンツを確立できていない、または独自のコンテンツで集客ができない
* SNS等でファンの獲得ができていない、知名度が十分にないイラストレーター
* どんなイラストに需要があるのかという観点から作品を制作したいイラストレーター
* 「自分の見たい作品」の制作を少額から支援することで実現させたいイラストファン
## 「プロダクト名」 
ReCo
## 「プロダクトのカテゴリー」 
マイクロクラウドファンディング
## 「顧客が得られる利益や体験」 
* クリエイターが得られる体験：
* 支援者＝完成を待つファンが複数名存在する状態で制作できることによる安心感/モチベーション向上
* 制作前に一定数の賛同と支援金が確定するため、需要のある作品の制作に集中できる。
* 知名度よりもリクエストに応えることができるという技術力が評価され、新規ファンの獲得につながる。
* ファンが得られる体験：
* 同じ趣向を持つ他ファンとの一体感
* 自分のアイディアや感性が肯定される満足感
* 少額の支援で自分のアイディアが形になる
## 「他の主要な競合製品」 「競合との主要な違い」 
- pixivFANBOX/既存のクラウドファンディングサービス
支援を受けるためにはSNS等での集客が必須である
- pixivリクエスト/SKIMA/Skeb等のイラストコミッションサイト
1対1の高額取引が基本
